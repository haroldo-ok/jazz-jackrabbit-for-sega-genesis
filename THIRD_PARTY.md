# Third-party references

- **OpenJazz** — https://github.com/AlisterT/openjazz — GPL-2.0-or-later.
  It was used as a technical/reference source for planning a future faithful
  JJ1 data/event import path. No OpenJazz source files are compiled into the
  present prototype.
- **SGDK** — https://github.com/Stephane-D/SGDK — development library and
  tooling used to build the Genesis ROM. It is an external build prerequisite.
- **Jazz Jackrabbit shareware data** — linked in `README.md`; not bundled.

The project is released under GPL-2.0-or-later to remain suitable for a future
OpenJazz-derived implementation. See `LICENSE`.
